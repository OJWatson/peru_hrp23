## pfhrp2/3 modelling in Peru

Repository containing analysis to assess the stochasticity of 
hrp2/3 frequency changes in Peru. 

`analysis` is the main analysis directory, which creates a `hipercow` 
environment to handle package dependency and set up for hpc usage. 

The full hipercow environment and raw model simulations is not committed
due to space, but the full package list and versioning is saved for 
basic reproducibility.
